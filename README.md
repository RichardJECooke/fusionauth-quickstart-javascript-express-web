## Prerequisites

Install Docker and Node.js.

## How To Run

In a terminal run the following to start fusionauth.

```shell
docker-compose up
```

In another terminal start the app.

```shell
cd complete-application
npm install
npm run start
```

Browse to the app at http://localhost:3000.
